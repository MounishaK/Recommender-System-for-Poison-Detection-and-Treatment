Minimized treatment time of a patient by developing a recommender system in C++ that detected 30 poison types. 
Introduced a data logging facility with a user-friendly interface.  




Steps for programming the code
=================================

Run the .cpp file
The headers are called in the main program

Choose the option from the main menu

Start with 1 for the diagnosis.

This initializes all data structures.

Enter the variable from the conclusion list 
i.e., SNAKE_BITE/ POISON/CHEMICALS
And perform yes/ no operations.

Then select option 2 for treatment.
